部署指南

1

选择部署平台

  • GitHub Pages - 免费，适合演示
  • Netlify - 自动部署，HTTPS
  • Vercel - 速度快，支持预览
  • 自行托管 - 需要服务器
2

获取访问URL

  • GitHub Pages: username.github.io/repository
  • Netlify: project-name.netlify.app
  • Vercel: project-name.vercel.app
  • 自定义域名: your-domain.com
3

生成二维码

  • 将URL输入上方生成器
  • 点击生成按钮
  • 下载或打印二维码
  • 分享给团队成员测试
